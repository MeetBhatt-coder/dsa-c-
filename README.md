 BANK MANAGEMENT SYSTEM PROJECT REPORT

1. Introduction
The Bank Management System is a software application developed using C++ that helps in managing basic banking operations. This system allows users to create accounts, deposit money, withdraw money, and view account details efficiently.
The project is designed using Object-Oriented Programming (OOPS) concepts such as encapsulation, inheritance, and polymorphism. It provides a simple and user-friendly interface through a menu-driven program, making it easy for beginners to understand and use.
This project demonstrates how real-world banking operations can be implemented using programming concepts.

2. Objectives of the Project
The main objectives of this project are:
•	To understand and implement Object-Oriented Programming concepts in C++
•	To develop a menu-driven banking system
•	To perform basic banking operations such as:
o	Account creation
o	Deposit money
o	Withdraw money
o	Display account details
•	To improve problem-solving and programming skills
•	To simulate a real-world application using C++

3. Methodology / System Design
The system is designed using a simple and structured approach:
 Classes Used:
•	Account Class
o	Stores account number, name, and balance
o	Contains functions for deposit, withdraw, and display
•	Bank Class
o	Inherits from Account class
o	Overrides display function (Polymorphism)
 Key Concepts Used:
•	Encapsulation → Data members are protected and accessed through functions
•	Inheritance → Bank class inherits Account class
•	Polymorphism → Function overriding using display()
•	Abstraction → Only necessary details are shown to the user
 Working:
1.	User selects an option from menu
2.	Program performs operation based on input
3.	Data is stored temporarily using arrays
4.	Output is displayed on screen

4. Implementation / Source Code
The project is implemented using C++ programming language.

The Bank Management System project successfully demonstrates the use of Object-Oriented Programming concepts in C++. It provides a basic understanding of how banking operations work in real life.
